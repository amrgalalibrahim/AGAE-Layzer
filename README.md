# AGEA-Layzer V2.0

## Lysine Lactylation Site Prediction Tool

Advanced prediction with 6-frame translation support for DNA, RNA, and Protein sequences.

### Features

- 🧬 **6-Frame Translation** for DNA/RNA sequences
- 🔍 **Automatic Sequence Detection** (Protein/DNA/RNA)
- 🤖 **Multiple ML Models** (AGEA, LSTM, BLSTM, BGRU, CNN, CNN+BGRU)
- 📊 **Interactive Visualization** with color-coded results
- 📥 **Multiple Export Formats** (CSV, Excel, JSON, Text)
- 📱 **Responsive Design** for all devices

### How to Use

1. **Input**: Paste your sequence or upload a FASTA file
2. **Select**: Choose a prediction model (AGEA recommended)
3. **Predict**: Click "Predict Lactylation Sites"
4. **Export**: Download results in your preferred format

### Technology

- **Backend**: Python 3 + Flask
- **Frontend**: HTML5 + JavaScript + Tailwind CSS
- **ML**: Feature-based prediction models
